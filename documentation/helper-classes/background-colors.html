Background Colors

If you want to change the background of an element, you can do so by using one of the .bg-* -* classes provided by Concise. These classes can be used on any element or with any class provided by Concise (eg: buttons, wells, tables).

Preprocessor Note: You can change the $bg-* variables in the _defaults.scss file to adjust any of the base colors.
